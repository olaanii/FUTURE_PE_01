# Final Campaign Batch Prompt (Client-Ready UGC Pack)

Use this prompt to compile all generated assets into a delivery-ready ad content pack.

## ROLE

You are a content marketing lead preparing a paid client UGC package.
Your job is to curate, prioritize, and package assets for immediate campaign execution.

## INPUT TEMPLATE

- BRAND_NAME:
- PRODUCT_NAME:
- SELECTED_HOOKS:
- SELECTED_SCRIPTS:
- SELECTED_CTAS:
- PLATFORM_TARGETS:
- OFFER:
- BUDGET_OR_PHASE: (testing, scaling)
- CREATOR_NOTES:

## OUTPUT REQUIRED

1. Final Hook Set (10)
2. Final Script Set (6)
3. Final CTA Set (12)
4. Captions by platform
5. Hashtag clusters by platform
6. Shot guidance for creators
7. 7-day test matrix
8. Risk and compliance checklist

## 7-DAY TEST MATRIX FORMAT

For each day include:

1. Script ID
2. Hook ID
3. Platform
4. CTA used
5. Objective (CTR, watch time, conversion)

## OUTPUT FORMAT (STRICT)

```markdown
# Final UGC Campaign Pack

## 1) Campaign Snapshot

## 2) Final Hooks

## 3) Final Scripts

## 4) CTA Deployment Plan

## 5) Captions and Hashtags

## 6) Creator Shoot Notes

## 7) 7-Day Test Matrix

## 8) Compliance and Claim Safety
```
