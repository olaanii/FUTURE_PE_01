# FUTURE_PE_02 - Prompt Engineering Task 2 (UGC Ad Content Marketing)

This project contains a reusable prompt system for creating high-converting UGC ad content using AI.

It is structured similarly to FUTURE_PE_01, but focused on paid social ad scripting workflows:

- hook generation
- short UGC scripts
- CTA banks
- tone and platform adaptation
- campaign-ready final pack

## Chosen Real Product (Example)

Mamaearth Vitamin C Face Serum (D2C skincare product in India)

You can replace the product in the input templates for any other real business.

## Folder Structure

- prompts/: reusable prompt systems
- outputs/: generated sample deliverables

## Suggested Workflow

1. Run prompts/master_prompt_system.md
2. Generate hooks using prompts/hooks_prompt.md
3. Generate scripts using prompts/ugc_script_prompt.md
4. Generate CTA bank using prompts/cta_prompt.md
5. Adapt for Reels/Ads/Shorts with prompts/platform_adapter_prompt.md
6. Lock tone variants with prompts/tone_adapter_prompt.md
7. Build final client-ready pack with prompts/campaign_batch_prompt.md

## Tools

- ChatGPT
- Claude
- Gemini
- Optional: CapCut, Runway, Pika for video execution
