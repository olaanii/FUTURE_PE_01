# One-Shot Build Prompt for Lovable or Framer AI

Paste this into Lovable or Framer AI after filling variables.

## Prompt

Build a premium, conversion-focused website for the following local business.
The client has no design reference, so you must make professional design decisions automatically.

### Business Input

- Business Name: {{BUSINESS_NAME}}
- Business Type: {{BUSINESS_TYPE}}
- Location: {{LOCATION}}
- Audience: {{TARGET_AUDIENCE}}
- Core Services: {{CORE_SERVICES}}
- Brand Personality: {{BRAND_PERSONALITY_WORDS}}
- Price Positioning: {{PRICE_POSITIONING}}
- Primary Goal: {{PRIMARY_GOAL}}
- Secondary Goal: {{SECONDARY_GOAL}}
- Trust Signals: {{TRUST_SIGNALS}}
- Primary CTA: {{PRIMARY_CTA}}
- Secondary CTA: {{SECONDARY_CTA}}

### Creative Direction Requirements

1. Use a modern, premium visual language suited to the business type.
2. Employ modern sophisticated UI skeletons: Bento-box CSS grids, asymmetric layouts, overlapping images with modern geometric masks (arches, pills), glassmorphism overlays, and floating pill-shaped badges/tags.
3. Do not use generic template layout.
4. Choose expressive typography: elegant display + highly readable body font.
5. Define clear design tokens: colors, spacing, radius, shadows, states.
6. Ensure mobile-first responsive behavior.

### Required Pages

1. Home
2. Services
3. Booking/Contact

### Required Home Sections

1. Hero with strong value proposition and dual CTA
2. Trust strip with concrete proof points
3. Service preview cards
4. Differentiator section
5. Objection handling FAQ
6. Final conversion CTA

### Required Services Sections

1. Services overview hero
2. Three service cards with outcomes
3. Process or "how it works" block
4. Trust reinforcement
5. CTA block

### Required Booking Page Sections

1. Booking hero
2. Structured booking form
3. Friction-reduction helper text
4. Trust + urgency side panel
5. Confirmation message state

### UX and Accessibility Requirements

1. Semantic headings and landmarks
2. Keyboard-focus visible states
3. High contrast text/background combinations
4. Touch-friendly controls on mobile
5. Clear form validation and success feedback

### Motion Requirements

1. Subtle entrance reveals (not distracting)
2. Meaningful hover and focus feedback
3. Smooth transitions under 250ms for micro-interactions

### Copy Constraints

1. Business-specific wording only
2. No fake claims or fake testimonials
3. Conversion-focused but human tone
4. Avoid robotic AI phrasing

### Output Needed

1. Fully designed pages with final copy
2. Reusable style system inside the project
3. Ready-to-publish structure
4. Clear CTA hierarchy on every page

### Final Quality Check Before Output

- Does this look like an agency-crafted professional site?
- Is the conversion path obvious within 5 seconds of landing?
- Is mobile UX polished and readable?
- Are sections visually distinct and purposeful?
