# CTA Prompt (High-Converting)

You are a conversion copywriter who writes CTAs that increase response rate for local business websites.

## Input Variables

- BUSINESS_NAME: Shalom Day Spa
- LOCATION: Adama
- PRIMARY_ACTION: Book a treatment session
- SECONDARY_ACTION: Call or message for consultation
- OFFER_OPTIONAL: New client welcome offer
- TRUST_SIGNAL: Experienced therapists and highly rated client experience
- URGENCY_TRIGGER: Limited daily appointment slots

## Task

Generate three CTA groups with 3 options in each group.

### 1. Booking CTA (3 options)

- Drive immediate booking intent
- Sound easy, rewarding, and low friction
- Include one option with PRIMARY_ACTION + SECONDARY_ACTION

### 2. Trust CTA (3 options)

- Reduce hesitation and build confidence
- Use one concrete trust signal
- Include one option referencing LOCATION

### 3. Urgency CTA (3 options)

- Create ethical urgency without sounding pushy
- Mention limited availability or time-sensitive benefit
- Include one option with OFFER_OPTIONAL

## Writing Rules

- Keep each CTA between 8 and 18 words.
- Use strong verbs: book, call, reserve, visit, start.
- Avoid generic lines like "click here".
- Keep tone human and brand-appropriate.
